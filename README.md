Restaurant Management System Overview
This terminal-based application is designed to streamline daily restaurant operations through a modular and efficient interface. It features a robust Table Management system that enforces capacity rules and tracks real-time occupancy across a custom 10-table layout.  
Key Capabilities:
• Order Lifecycle Management: Facilitates the entire dining workflow, from seating guests and tracking active orders to calculating final bills with tax and splitting options.  
• Dynamic Menu Control: Provides administrative tools to modify menu items, prices, and categories instantly without restarting the system.  
• Data Persistence: Utilizes a reliable JSON-based storage system to automatically save the state of tables, menus, and orders, ensuring no data is lost between sessions.  
• Modular Architecture: Built with clean separation of concerns (Tables, Menu, Orders, Storage), ensuring the code is maintainable and scalable.
